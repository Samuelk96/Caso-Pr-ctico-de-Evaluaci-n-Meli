# Caso-Pr-ctico-de-Evaluaci-n-Meli
Solución de BI diseñada para automatizar el monitoreo semanal de la apertura de 12 Service Centers en México. Reduce el tiempo de preparación de +2 horas a menos de 5 minutos mediante ingeniería de datos con Python, análisis predictivo de riesgos con IA (Claude) y tableros de control ejecutivos en Looker Studio.
